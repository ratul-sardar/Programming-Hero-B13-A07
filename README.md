👥 KeenKeeper — Keep Your Friendships Alive
📌 Overview

KeenKeeper is a responsive web application designed to help users maintain meaningful relationships by tracking interactions with friends. It allows users to monitor contact frequency, log interactions (Call, Text, Video), and visualize communication patterns through analytics.

⚡ Interaction Tracking
Log Call, Text, and Video interactions instantly

Centralized interaction history
Filter entries by interaction type (Call / Text / Video)
